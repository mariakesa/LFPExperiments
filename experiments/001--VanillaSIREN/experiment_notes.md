# Vanilla SIREN fit

## Experiment goal

The goal is to fit a vanilla SIREN model to Neuropixel probe data and check how well it approximates
the data on the test set. 

## Research questions

What is a possible architecture to get the fit to work (how large the network has to be for a particular dataset size)?

## Exploratory analysis

Observation of wave-like phenomena.
